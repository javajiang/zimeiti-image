# Zimeiti Image

面向电商从业者的 Obsidian 插件，用于收集竞品电商图片，提取风格，并生成新的电商主图或详情图。

## 适用人群

- 电商运营
- 视觉设计师
- 商品详情页制作人员
- 需要分析竞品主图、详情图的从业者

## 核心能力

- 单张图片风格蒸馏
- 多张图片风格蒸馏
- 基于风格生成图片
- 风格结果保存到 `图片风格库`
- 生成结果保存到 `生成图片`

## 目录约定

- `图片素材`：放待分析的竞品图片
- `图片风格库`：保存蒸馏后的图片风格文件
- `生成图片`：保存生成结果

## 安装与启用

### 第一步：打开 Obsidian 后，点击左下角添加 Vault

选择你收到的完整 Vault 文件夹。

![第一步-添加Vault](images/step1-add-vault.png)

### 第二步：点击“信任作者”

如果 Vault 内包含插件和配置，Obsidian 会弹出安全提示。这里需要点击“信任作者”。

![第二步-点击信任](images/step2-trust-author.png)

### 第三步：点击插件右侧按钮，开启插件

在社区插件列表中找到图片插件，点击右侧开关启用。

![第三步-开启插件](images/step3-enable-plugin.png)

### 第四步：点击插件设置，输入 API Key

进入插件设置页，在 `API Key` 输入框中填入你的 Key 并保存。

![第四步-输入APIKey](images/step4-input-api-key.png)

### 第五步：查看插件使用说明

启用成功后，可以打开 Vault 内的图片插件说明文档查看完整流程。

![第五步-查看使用说明](images/step5-open-docs.png)

## 使用方式

1. 在插件设置中填写 API Key。
2. 把竞品图片放到 `图片素材` 或业务文件夹中。
3. 打开一张或多张图片。
4. 右键执行 `图片风格蒸馏`。
5. 输入风格名称。
6. 在文件夹上右键选择 `生成图片`。
7. 填写生成需求，选择是否使用已有风格。
8. 结果会保存到 `生成图片`。
